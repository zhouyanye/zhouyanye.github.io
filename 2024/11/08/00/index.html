设备 1：FW1（防火墙）

1. 配置接口 IP 地址并加入 DMZ 区域

  1. 进入防火墙配置模式。
  2. 在防火墙上配置接口 GE0/0/3 的 IP 地址，并将接口加入 DMZ 区域。

2. 配置 NAT 服务器

NAT 服务器配置会将外部访问 1.1.1.1 的 HTTP 请求映射到内部服务器 192.168.6.2 的 HTTP 服务上。

3. 配置防火墙策略

6. 检查日志（调试用）

如果配置完成后仍然无法访问，可以检查防火墙日志来确认流量是否被阻挡。
